# Requerimientos - Asgard Pets v1.0


## Descripción del proyecto

Asgard Pets es una empresa ficticia que se encarga de la distribución y venta de alimento, medicamentos y accesorios para mascotas. Asgard busca promover la salud y bienestar de las mascotas desde la alimentación y el buen cuidado. 

Las mascotas ya hacen parte de la familia de cada Colombiano, por lo cual nuestro énfasis es darle cuidado, protección y amor a las mascotas, por este motivo Asgard Pets creó la fundación Aurora, Asgard utiliza una parte de sus ganancias para el sostenimiento de la fundación. 

## Problema que resuelve

Actualmente Asgard utiliza un inventario en Excel el cual no es eficiente y muchas veces genera errores. 
## Objetivo

Crear un sistema que sea eficiente para la gestión de los productos de Asgard.

## Usuario del sistema

El programa en está versión será usado por un empleado de Asgard para que gestione el inventario.

## Funcionalidades

Para la versión 1.0 de Asgard Pets el sistema debe:

- Permitir registrar productos.

- Permitir buscar un producto por su nombre.

- Mostrar el inventario.

- Actualizar la cantidad de un producto.

- Calcular el valor total de un producto.

## Restricciones

- Esta versión solo tendrá entrada por consola.

- No tendrá una base de datos, la información solo será almacenada en Arrays.
- Cuando se cierre el programa los datos no quedarán guardados.
- Se utilizará Java básico, sin POO ni frameworks.
